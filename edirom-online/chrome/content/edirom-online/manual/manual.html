Help (User Manual)

This document provides a general introduction to using Edirom Online. It describes the basic functions of the software. Please note that not all of the options described here are necessarily available in every edition using Edirom Online. This is due to the fact that contents have to be prepared accordingly and projects using Edirom Online have their own edition guidelines and concepts. Furthermore, some versions may contain more options than mentioned in this document due to edition-specific enhancements.

User interface

The user interface of Edirom Online is divided into several sections. At the top and bottom borders of the window you find Toolbar and Taskbar that provide access to all major Edirom functions. The space in between is called Desktop . At the top right corner of the Desktop is the so-called Navigator .

Figure 1. Overview of the Edirom online screen, showing the Desktop with Toolbar (top), Taskbar (bottom), Navigator (right), and Concordance navigator.

Toolbar

Figure 2. Detail view of the Toolbar.

On the left side of the Toolbar the Home button is located. At the very right there is an icon for opening a Search window, which permits a text-based Search within the edition.

Home button

The Home button reopens the startup dialog.

Search

Clicking the magnifying lens icon opens the Search .

Navigator

At the top right corner of the Desktop, right below the Search, there is the so-called Navigator containing a list of contents that are specific for each edition.

Desktop

The Desktop is the space between Toolbar and Taskbar. In this space all internal application windows are opened and arranged in order to show contents of an edition or to give access to further application functions. The internal application windows cannot be dragged out of this area. See Content window (content object) for further details.

Taskbar

Figure 3. Detail view of the Taskbar.

The Taskbar at the bottom of the Desktop gives access to content-related functions on the left, and meta-functions on the right.

Sort windows in grid

This button allows arranging all open windows in tiles of a grid.

Sort windows horizontally

This buttons allows arranging all open windows in horizontal rows.

Sort windows vertically

This buttons allows arranging all open windows as columns.

Show measure numbers

The Show measure numbers button permits to show or hide bar numbers in all windows globally. See Content window (content object) for further details.

Show annotations

This button allows to show or hide annotation icons globally in all windows. For further details see Content window (content object) .
Annotation filter settings (see Filtering ), however, can only be configured in the Window toolbar of each individual window.

Open concordance navigator

This button opens or closes the Concordances window.

Help

You can use this button to display this Help (Edirom User Manual) window or to move it to the foreground.

Search

The Search window can be opened by clicking the Search (magnifying lens) on the Toolbar. It will perform a text-based search on all edition contents.

Figure 4. Screenshot of the Search window after performing a search operation.

After performing a search operation, all resulting hits are grouped as one result entry for every corresponding content object (text document, annotation, etc.). Furthermore, individual hits are highlighted within their surrounding context.

Clicking on an entry's title or on a highlighted search hit opens the corresponding content object with all search hits highlighted.

If a search operation scores more than three hits per entry a link named ‘show all hits’ will appear.

Navigator

The Navigator offers quick access to the objects contained in the edition, which are opened by mouse click in a new Content window (content object) on the current Desktop .

Figure 5. Screenshot of the navigator. Note the scrollbar on the right.

The number of objects held by the Navigator may exceed the actual size of the panel. In this case a scrollbar at the right side of the window permits to scroll through the contents of the Navigator. The mouse wheel may be used as well.

Although the Navigator is fixed to the top right corner of the workspace, it is possible to resize and even to reduce it to a narrow bar below the Search button by drag-and-drop on its left or bottom edge.

Concordances

The Concordances controls synchronous navigation to corresponding places in all open source or edition windows. It is based on the structural information (e.g., numbers, scenes, bars, text lines) inherent to the source and edition objects.

The Concordances consists of one or two drop-down menus, depending on wether the selected number/scene in the upper drop-down menu provides multiple options for navigating through the edition and/or sources. Navigation by bar as the default option allows displaying and navigating through all corresponding occurrences of a particular bar synchronously. Furthermore, if provided, the second drop-down menu allows to select Navigation by text line, which allows displaying and navigating by referring to corresponding text lines.

A particular bar or text line may be selected by using the slide control. By clicking the Display button, all corresponding edition or source windows get synchonized. Browsing backward or forward simultaneously within the open sources and editions is achieved using the two arrows to the left and right of the Display button.

Content window (content object)

Any content object contained in the edition opens in a separate window, called Content window. The Content windows can be manipulated within the program in virtually the same way as normal operating system windows.

Windows can be arranged on the Desktop by dragging the title bars; they may be resized by dragging any edge or corner of the window frame.

Each open window is represented by a button in the Taskbar showing the window title, a feature that might be known from certain computer operating systems. Clicking on the button allows toggling between minimizing and extending window function.

When multiple windows are open, the entry in the Taskbar indicates which window is currently active.

Title bar

Along the top edge of the window is a title bar that contains the name of the displayed object.

At the right end of the title bar there are three buttons:

(or a click on the respective entry in the Taskbar) minimizes the window to the Taskbar.
(or a click on the respective entry in the Taskbar) extends the window to the full size of the current Desktop.
(or ‘Esc’) closes the window.

Window toolbar

Below the title bar each Content window has a Window toolbar. At the very left of the Window toolbar there is a combo box for switching between Views. Further content specific features may be accessible through the Window toolbar, such as Display , Annotations (Readings, Critical Report) , Show/Hide measures , and Go to… .

Window taskbar

At the bottom edge Content windows may contain a view specific Window taskbar with tools for manipulating the object displayed in the window. The availability is dynamic and depends on the object currently on display. Specifically, in windows showing images (facsimiles) of musical sources or editions the Window taskbar includes buttons for toggling between the two display modes, i.e. the Page based view and the Measure based view.

Views

Views offer different perspectives on the content of the currently open window, e.g., Text, Facsimile, and XML-View . In certain cases, other views (such as Synopsis, Preliminary Remarks, or Source Description) may be included.

Text

The entry Text is available on a window's Window toolbar menu if the content object of the window is a text-based object, e.g., a TEI file representing the Preface, the Editorial Guidelines, the Text Edition, the List of Sources, or the Source Evaluation. The Text appears in a form similar to that of a website and may include links to other content (depending on the edition, the appearance of the actual link may vary).

Facsimile

The Facsimile view shows images of a source or an edition and provides controls for navigating in it.

The Window toolbar of Content windows displayed in Facsimile view may contain the following additional control menus:

Display
allows configuration of basic Display options such as the display of text versions.
Annotations
allows configuration of superimposed Annotation symbols. The Annotations drop-down menu provides access to the following functions:
Show/hide annotations
This option permits to display Critical Report Annotations shaped as buttons superimposed on facsimiles, images, or text. The user can filter Annotations by category (see below). In each source or edition window, only Annotations relating to that particular source or edition are shown. If you mouse over an annotation symbol, a tooltip shows preview information concerning the annotation. Click on the symbol to open the annotation in its separate window (for a detailed description see Annotations (Readings, Critical Report) ).
Categories…
This sub-menu shows all categories (if any) of Annotations contained in the current source or edition and allows activating/deactivating their display in the Facsimile view.
Show/Hide measures
This option allows to control the superimposing of bar numbers for the source or edition on display. Bar numbers appear in the center of the corresponding bar.
Go to…
allows skipping to specific sections such as numbers and scenes. In contrast to the Concordances , the Go to… function affects only the currently active Content window.

The Window taskbar of Content windows displayed in Facsimile contains the following controls for navigating the source or edition images:

Page based view
switches to a page based navigation of the source or edition. The Page based view displays the source or edition page by page, including a Zoom Slider and the Fit facsimile button.
Measure based view
switches to a bar based navigation of the source or edition. The Measure based view displays the source or edition bar by bar. In this mode, the Window taskbar provides combo boxes and input fields required for navigating to any place in the respective source or edition (e.g., number or scene, bar, bar range; for details see below).
Zoom Slider
The Zoom Slider slider allows zooming in and out of the image currently displayed.
The zoom level can also be increased or decreased using the scroll function of the mouse wheel.
Fit facsimile
fits the current page into the window.
Page Spinner (in the Page based view), or Measure Spinner (in the Measure based view)
Insert a page or bar number, or click on the respective arrows for navigating to the desired place in the source or edition. When entering a number in the number field, a filtered list of possible numbers will appear. The numbering corresponds to the one applied by the editor to the specific source or edition; it does not necessarily begin by 1. Furthermore, it might include any alphanumeric character.

Additional controls within the Measure based view include:

Movement combo box
Allows selecting the number and scene.
Select parts
Allows selecting the parts to be displayed. This control is available if the current source consists of a set of parts (e.g., orchestral or choral parts).
Measure range
This input field allows entering the number of consecutive bars to be displayed.

XML-View

Selecting XML-View from the View menu shows the XML data (e.g. the MEI or TEI encoding) of the current content object. In this view, the following controls are present in the Window toolbar:

A-
Decrease font size button.
A+
Increase font size button.
Line #
The Show line numbers button enables to show and hide the line numbers on the left hand side of the window.

Annotations (Readings, Critical Report)

In Edirom Online, Annotations within an edition's Critical Report can be accessed in different ways.
(1) Icons representing Annotations in the editions and sources provide essential information via tooltip (mouseover).
(2) A click on the icon opens the full view of the Critical Report entry.
(3) A list view of Annotations contained in the Critical Report is also accessible through the Navigator .

Moreover, Edirom Online provides additional means for accessing individual Annotations by labelling them according to specific Categories.

Figure 7. Screenshot of the edition with annotation icons and tooltip of a single annotation.

Views

List view

The Annotations window shows readings or individual Annotations from the Critical Report of the current edition. The default appearance is a list of all Annotations with three columns:

No.
Indicates the order of the annotations.
Title
Gives the heading or lemma of the annotation; often used to give a first impression of the annotation's subject matter, or to indicate the position of an annotation.
Categories
Indicates the Categories assigned to a given annotation.

Clicking on a column header will re-sort the list on the basis of this column’s data, a second click on the same column header will reverse the sort order. When moving the mouse over the column header an arrowhead will appear to the right of the column title, giving access to a context menu that enables:

setting sort order, and
showing/hiding columns.

It is also possible to apply a filter on the respective column. For more details, see Filtering .

Figure 8. List of Annotations showing options provided in a column's context menu.

Individual annotation view

Double-clicking on an Annotation entry in the list switches to a full view of all the individual Annotation. It presents, along with the text and further identifying parameters, snippet views of all related sources and editions in an overview grid; double-clicking on a single image opens the corresponding source in its Content window (content object) .

The Window taskbar of the Individual annotation window shows the following controls:

List view
Returns to the complete list of annotations.
Previous annotation
Moves to the previous annotation with respect to filters and sort options applied in the list of annotations (see List view ).
Next annotation
Moves to the next annotation with respect to filters and sort options applied in in the list of annotations (see List view ).
Open all sources
Opens detail views of all sources corresponding to the current annotation.
Close all sources
Only available when Open all sources has been operated before, closes all respective windows.

Filtering

Filtering annotations by text contained in the title (heading, lemma, position) as well as by Categories is possible from the list options (context menu) in the Views of the Annotations windows. This filtering concerns only the Annotations object (in both Categories and Individual annotation view).

In any Content window (content object) , the display of Annotation buttons can be filtered by Categories. This function is available only from the menu ‘Annotations’ in Facsimile . All Annotations that belong to at least one of the selected Categories are shown in the facsimile or edition. Please note that Filtering by multiple Categories will display all Annotations assigned to at least one of these Categories.

Help (Edirom User Manual)

The Help (Edirom User Manual) window is (in contrast to content related windows) a stand-alone window and can only be opened once. Clicking the Help button will toggle show/hide for the help window, or will bring it to the foreground if hidden behind other windows.
